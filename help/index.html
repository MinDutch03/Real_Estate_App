B) Directories & File Uses

Our Design uses only assets & lib folder. So, don't touch any other folder for ui changes.

    • assets(Folder) - Put all Static images & other data (in Our Case images) in this folder
    • fonts(Folder) - Put all font file in this folder
    • lib(Folder) - Main folder that contains all the dart code
      • constants(Folder) - Contain Constants (fixPadding, textstyle, etc) in this folder
      • pages(Folder) - Contain All App Pages in this folder except main.dart
        • add_new_listing(Folder) - Contain add new listing page in this folder
          • add_new_listing.dart(Page) - Add new listing screen code.
        • auth(Folder) - Contain All authentication related file in this folder
          • login.dart(Page) - Login screen code.
          • otp.dart(Page) - Otp screen code.
          • register.dart(Page) - Register screen code.
        • chat(Folder) - Contain all chat related file in this folder
          • chat_list.dart(Page) - Chat list screen code.
          • chat.dart(Page) - Chat screen code.
        • home(Folder) - Contain home page in this folder
          • home.dart(Page) - Home screen code.
        • my_listing(Folder) - Contain my listing page in this folder
          • my_listing.dart(Page) - My listing screen code.
        • notifications(Folder) - Contain notification page in this folder
          • notifications.dart(Page) - Notifications screen code.
        • property(Folder) - Contain property page & full screen image in this folder
          • full_screen_image.dart(Page) - Full screen image screen code.
          • property.dart(Page) - Property screen code.
        • search(Folder) - Contain search page in this folder
          • search.dart(Page) - Search screen code.
        • settings(Folder) - Contain settings related file in this folder
          • edit_profile.dart(Page) - Edit profile screen code.
          • privacy_policy.dart(Page) - Privacy policy screen code.
          • settings.dart(Page) - Settings screen code.
          • support.dart(Page) - Support screen code.
          • terms_of_use.dart(Page) - Terms of use screen code.
        • shortlisted(Folder) - Contain shortlist page in this folder
          • shortlisted.dart(Page) - Shortlist screen code.
      • bottom_bar.dart(Page) - Contain bottom bar in this file.
      • screens.dart(Page) - Export all screens in this file.
      • splashScreen.dart(Page) - Contain splash screen code.
    • widget(Folder) - Contain all our created widget (ColumnBuilder & RowBuilder).
      • column_builder.dart(Widget File) - Code of ColumnBuilder widget.
      • row_builder.dart(Widget File) - Code of RowBuilder widget.
    • main.dart - Entry Point of Our App.
  • pubspec.yaml(File) - Specification file (All the package and local assets (Like Images, Fonts, etc.) we use in this application is specify in this file.).
  • Packages (Dependencies)
  • Assets
  • Fonts

C) About Google Map Configuration

We use google_maps_flutter package for google map.

You need Google Map Api key for Accessing Google Map.

  • If you want to know how to get Google Map Api Key then please visit this link google_maps_flutter.

    • 1)Android

    • Put that api key in 1 places in our code.
      • my_properties/android/app/src/main/AndroidManifest.xml line number 42. (Please refer below image)

    2) iOS

    • Put that api key in 1 places in our code.
